# What is Physical-AI ?

Physical AI is the next-gen technology that involves integration of artificial intelligence with embodied physical systems such as robots, autonomous machines, sensors, and cyber-physical infrastructure in order to enable real-world perception, reasoning, interaction, and adaptive autonomous action within dynamic environments.
